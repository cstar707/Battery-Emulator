All inverter controls and buttons are on this page. Use the sections below to change mode, export, charging, and storage/hybrid bits.

Work Mode

Primary Solis work-mode controls. Use these buttons to switch modes without manually toggling raw bits.

These four modes are mutually exclusive — switching to one clears the other three in a single atomic write.

Self-Use Mode {% if storage_bits.self_use %}On{% else %}Off{% endif %}

Solar → load → battery. Use local solar first.

Feed-In Priority {% if storage_bits.feed_in_priority %}On{% else %}Off{% endif %}

Export solar first when available.

Peak-Shaving Mode {% if storage_bits.peak_shaving %}On{% else %}Off{% endif %}

Limit grid draw during peak load conditions.

Off-Grid Mode {% if storage_bits.off_grid %}On{% else %}Off{% endif %}

Prioritize isolated operation away from the grid. Forces Export and Grid Charge off.
