# Battery-Emulator Setup Notes

This is a minimal local guide based on the upstream README and wiki. Always prefer the upstream instructions for latest steps and hardware compatibility.

## Quickstart (Upstream Summary)
- Pick a supported inverter
- Pick a supported battery
- Order compatible hardware
- Follow installation guidelines for your battery and inverter

## Software Install (Web Installer)
1. Open the web installer
2. Follow the on-screen instructions to flash the device
3. Connect to the Battery-Emulator Wi-Fi (default password in upstream README)
4. Open the setup page and configure component selection
5. Optionally connect the board to your home Wi-Fi
6. Connect the battery and inverter to the board
